¿Qué es Qt?

  • Cross-platform Framework
    • Desktop Platforms
      • Windows, Linux/X11, Mac OS X
    • Mobile Platforms
      • Android, iOS, WinRT
    • Embedded Platforms
      • Embedded Android, Embedded Linux, Windows Embedded
  • Usado para crear aplicaciones con Interfaz Gráfica de Usuario
  • No está limitado sólo a la GUI
    • Networking, Database, OpenGL, Multimedia, QML/Qt Quick, WebKit
  • Licencias
    • Libre
    • Comercial
  • Herramienta más usada

¿Qué es PyQt?

Signals y Slots

Comunicación entre objetos

Sintáxis


boton = QPushButton("Clickeame!")
					

def mi_funcion():
    print("Botón clickeado")
					

# Qt, conectáme la señal 'clicked' del botón a la función 'mi_funcion'
boton.clicked.connect(mi_funcion)

Slot con argumentos


def mi_otra_funcion(nombre):
    def print("Hola {name}!".format(name=nombre))
    					

boton.clicked.connect(lambda: mi_otra_funcion('Gabo'))
